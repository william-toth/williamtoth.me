CodeNames | Github | Web App (in progress)

Codenames is a fun 4+ person game that I played with my friends often in high school. After I learned HTML/CSS/JS, Node, and Express, I realized I could make my own Codenames web app (server currently in progress), which is exactly what I am doing. In Codenames, each team (red and blue) has a spymaster and numerous players. The spymaster, who can see the word colors will say a word and a number, the word connecting that teams' words on the grid and the number specifying how many it connects to, and it is up to the players to guess which words are theirs. The first team to get all of their words wins!

Search Engine | Github

This Search Engine was the culminating project of my Software Development course. It contains a crawler, which, given a URL and depth, crawls the web, saving html files, an indexer, which reads saved files, creating an inverted index data structure of words and their number of occurences in each html file, and a querier, which, given a query, will use the inverted index data structure to search the web and subsequently return URLS matching the query. The above output is the result of querying on an index creating by crawling our course example website.

Data Science Project: Evolution of Popular Music | Github

In this project, I analyze data from every song in the Billboard Hot 100 from 1960 - 2010 in an attempt to understand which quantitative music features contribute to shifts in the musical zeitgeist. Through the random decision forest algorithm, I then predict the era and style of a given song based upon 16 quantitative musical features.

Markov Model: Part of Speech Tagger | Github

This project uses hidden markov modeling and the viterbi algorithm to predict part of speech given a body of text. Currently, it is set to train the algorithm with the Brown Corpus, but you can train it with any two text and POS files.
